# v0.9.41 — Flexible firmware matching + Split42

> Protocol unchanged at **v11** — pairs with firmware 0.9.67.

### Connects across firmware versions 🔗
No more whole-menu lockout on a version mismatch: the host connects to any compatible firmware and disables only the individual features a given keyboard is too old for. If the firmware is *newer* than the app, it prompts — Safe mode / Check for updates / Connect anyway.

### Split42 support ⌨️
Recognizes the Corne-size Split42 (auto-detect + Linux udev access), making it a first-class device alongside the Split72.

### Eden idle style 🌌
Pick the "Eden" comet-field screensaver from the tray (and `polyctl`), with a faithful on-screen preview and a "Reset Eden" replay button.

### Native file dialogs on Linux 📂
Uses the native KDE/Plasma file picker (portal) for the overlay-mapping and font-pack dialogs, defaulting to Downloads; the standard Qt dialog elsewhere.

### Font-pack inspector & editor 🔤 (power users)
A tool (under `--debug`) to inspect every font-pack bundle glyph as the keycap draws it, and to build/splice new glyphs from a TTF with a real-keycap OLED preview.

*Plus a forwarder startup fix, self-update/release tooling, and docs/maintenance across 0.9.27–0.9.41. 🛠️*
