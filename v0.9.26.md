# v0.9.26 — DOOM install, expanded scripts & About dialog

> Advances the host protocol to **v11** — update the firmware (0.9.44) to match or you'll see *"Protocol mismatch."*

### 0.9.25 — About dialog + diagnostics ℹ️
New About dialog with version, live keyboard metrics (uptime, overlay count, paths), one-click Copy-diagnostics, and project/support links.

### 0.9.22 — Bring your own DOOM 📦🔥
New `polyctl doom install` / `install-pack` flashes the DOOM game data and engine pack onto the keyboard over HID — everything the hardware needs to run it. Also: the HID console now self-heals after a firmware apply, and a "Success" log flood is fixed.

### 0.9.21 — Overlay upload fix (protocol v11) 🔌
Matches the firmware's repacked plain-overlay header. **Requires matching firmware — update both.**

### 0.9.17–0.9.18 — 9 more keycap scripts (protocol v10) 🔤
Host support for the expanded glyph-script set (runes, Aurebesh, SGA, Cirth, IBM VGA, C64, Amiga, APL, Braille), auto-flashed as the fantasy font-pack bundle, with smoother Braille/APL rendering. **Protocol v10 — update both.**

*Plus self-update/stability fixes and maintenance across the rest. 🛠️*
